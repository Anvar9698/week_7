# Week 7 - Abstraction (Java OOP)

## Overview
This project demonstrates the concept of **Abstraction** in Java using an abstract class and multiple subclasses. Each appliance inherits common properties and methods while implementing its own specific operation.

## Concepts Used
- Abstract Class
- Abstraction
- Inheritance
- Method Overriding
- Polymorphism

## Classes
- Appliance (Abstract Class)
- AirConditioner
- WashingMachine
- Microwave
- Television
- Refrigerator
- Main

## How to Run
1. Compile the Java files.
2. Run the `Main` class.
3. The program will display the behavior of different appliances.
